# falcon-plugins

Claude Code プラグインのマーケットプレイス。

## プラグイン一覧

| Plugin | Description |
|--------|-------------|
| [best-practices](https://github.com/kae-builds/best-practices) | ソフトウェア開発ベストプラクティス |
| [notion-tools](https://github.com/kae-builds/notion-tools) | Notion API直接連携 |

## インストール

```bash
# マーケットプレイス追加
/plugin marketplace add kae-builds/falcon-marketplace

# プラグインインストール
/plugin install best-practices@falcon-plugins
/plugin install notion-tools@falcon-plugins
```
